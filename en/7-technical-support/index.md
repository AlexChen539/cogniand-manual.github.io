## When Submitting Issues, Please Provide

To resolve issues faster, please provide the following information:

### 1. Problem Description
- Detailed description of the problem
- Time and frequency of occurrence

### 2. Steps to Reproduce
- Detailed steps to reproduce the problem
- Expected result vs actual result

### 3. Environment Information
- Operating system (Windows/Mac/Linux)
- Browser type and version
- Account role (subject/experimenter/admin)

### 4. Error Information
- Complete error message
- Browser console error messages (if any)
- Screenshots or screen recordings (if applicable)

## How to Get Browser Console Errors

If you encounter technical issues, viewing browser console can help diagnose:

**Open Console**:
- Chrome/Edge: `F12` or `Ctrl + Shift + I`
- Firefox: `F12` or `Ctrl + Shift + K`
- Safari: `Cmd + Option + I` (need to enable developer menu in preferences first)

**View Error Messages**:
1. Open console
2. Switch to "Console" tab
3. Look for red error messages
4. Take screenshot and send to technical support

## Security Vulnerability Reporting

If you discover a security vulnerability on the platform, please:

1. **Do Not Publicly Disclose**
   - Do not publish vulnerability information in public channels
   - Avoid malicious exploitation

2. **Contact Technical Team**
   - Send email to: security@cogniand.com
   - Describe vulnerability situation in detail
   - Provide reproduction steps (if any)

3. **Wait for Response**
   - We will respond within 24 hours
   - Will fix promptly after confirming vulnerability
   - Will notify you after fix
