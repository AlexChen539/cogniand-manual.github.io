# 社交分享

## 概述

社交分享功能允许您将邀请链接快速分享到社交媒体平台，扩大邀请的覆盖面。通过社交媒体分享，您可以触达更广泛的潜在被试群体，提高邀请的传播效率。

## 进入社交分享

登录后，进入"邀请用户"页面，在页面左侧"分享邀请链接"区域找到社交媒体分享按钮。

## 功能说明

### 1. 支持的社交平台

#### 1.1 Twitter

**平台特点**：
- 全球性社交媒体平台
- 适合学术和专业交流
- 支持话题标签传播

**分享方式**：
1. 点击Twitter分享按钮（蓝色按钮）
2. 自动打开Twitter分享对话框
3. 预填分享文本："加入我们的认知科学研究平台！{邀请链接}"
4. 可以编辑分享文本
5. 点击"Tweet"发布

**使用建议**：
- 添加相关话题标签（如#心理学研究 #认知科学）
- 简要说明实验内容
- @相关的学术账号或机构

#### 1.2 LinkedIn

**平台特点**：
- 专业社交网络平台
- 适合学术和职业交流
- 用户群体专业性强

**分享方式**：
1. 点击LinkedIn分享按钮（蓝色按钮）
2. 自动打开LinkedIn分享对话框
3. 自动填充邀请链接
4. 可以添加分享说明
5. 点击"Post"发布

**使用建议**：
- 说明研究的专业价值
- 强调参与的学术意义
- 针对专业人士撰写内容

### 2. 分享操作

#### 2.1 点击分享按钮

**操作步骤**：
1. 在邀请页面找到社交分享按钮
2. 选择要分享的平台（Twitter或LinkedIn）
3. 点击对应的分享按钮
4. 在弹出的窗口中完成分享

**窗口特点**：
- 在新窗口打开（600x400像素）
- 使用各平台的官方分享API
- 自动填充邀请链接
- 支持编辑分享内容

#### 2.2 编辑分享内容

**可编辑内容**：
- 分享文本
- 话题标签
- @提及的用户
- 附加说明

**编辑建议**：
- 保留邀请链接
- 添加吸引人的描述
- 使用相关的话题标签
- 保持内容简洁明了

#### 2.3 完成分享

**发布后**：
- 关闭分享窗口
- 返回邀请页面
- 分享内容出现在您的社交媒体账号上
- 您的关注者可以看到邀请链接

### 3. 分享内容优化

#### 3.1 文案建议

**吸引注意**：
- 使用简洁有力的开头
- 突出实验的独特性
- 强调参与的价值

**示例文案**：
```
🧠 诚邀参与认知科学研究！

我们正在进行一项关于工作记忆的在线实验，
只需20分钟，即可了解您的认知特点。

参与即可获得：
✅ 个性化认知报告
✅ 为科学研究贡献力量
✅ 平台积分奖励

点击链接注册参与：{邀请链接}

#心理学研究 #认知科学 #实验招募
```

#### 3.2 话题标签

**推荐标签**：
- #心理学研究
- #认知科学
- #实验招募
- #被试招募
- #学术研究
- #在线实验

**使用技巧**：
- 选择相关且热门的标签
- 标签数量建议3-5个
- 避免使用过于宽泛的标签

#### 3.3 视觉元素

**增强吸引力**：
- 使用表情符号（如🧠💡📊）
- 使用项目符号列表
- 突出关键信息
- 保持排版整洁

### 4. 分享效果追踪

#### 4.1 查看分享数据

**追踪方式**：
- 通过社交媒体平台的分析工具
- 查看帖子的浏览量、点赞数、转发数
- 分析哪些内容更受欢迎

#### 4.2 优化分享策略

**数据分析**：
- 记录不同文案的效果
- 分析最佳发布时间
- 了解目标受众的偏好
- 持续优化分享内容

## 使用建议

### 1. 选择合适的平台

**Twitter适合**：
- 快速传播信息
- 触达广泛受众
- 学术交流和讨论

**LinkedIn适合**：
- 专业人士招募
- 学术合作推广
- 职业相关研究

### 2. 最佳发布时间

**Twitter**：
- 工作日上午9-11点
- 下午1-3点
- 晚上7-9点

**LinkedIn**：
- 工作日上午7-9点
- 中午12-1点
- 下午5-6点

**注意事项**：
- 考虑目标受众的时区
- 避免在深夜或凌晨发布
- 周末发布效果可能较差

### 3. 提高分享效果

**内容质量**：
- 提供有价值的信息
- 使用专业的语言
- 保持真实和透明

**互动参与**：
- 及时回复评论和私信
- 感谢转发和支持
- 解答潜在被试的疑问

**持续推广**：
- 定期分享更新
- 分享实验进展
- 感谢参与者的贡献

## 注意事项

⚠️ **重要提示**：

1. **平台规则**
   - 遵守各社交平台的使用条款
   - 不要发送垃圾信息
   - 不要过度频繁发布

2. **内容真实性**
   - 不要发布虚假信息
   - 不要夸大实验效果
   - 不要承诺无法兑现的奖励

3. **隐私保护**
   - 不要泄露被试的个人信息
   - 不要分享未经授权的数据
   - 尊重参与者的隐私权

4. **专业形象**
   - 保持专业的语言和态度
   - 避免使用不当的表情或图片
   - 维护学术研究的严肃性

5. **版权问题**
   - 不要使用未经授权的图片或内容
   - 注明引用来源
   - 尊重知识产权

## 常见问题

### Q1: 为什么分享窗口打不开？

**A**:
- 检查浏览器是否阻止了弹出窗口
- 在浏览器设置中允许弹出窗口
- 尝试刷新页面后重试

### Q2: 可以同时分享到多个平台吗？

**A**:
- 可以，分别点击不同平台的分享按钮
- 每个平台需要单独操作
- 建议根据平台特点调整分享内容

### Q3: 分享后可以删除吗？

**A**:
- 可以，在对应的社交媒体平台上删除帖子
- 删除后链接仍然有效
- 已经看到的用户仍可以访问链接

### Q4: 分享内容可以自定义吗？

**A**:
- 可以，在分享对话框中编辑内容
- 建议保留邀请链接
- 可以添加个性化的描述和标签

### Q5: 如何知道有多少人通过社交分享注册？

**A**:
- 查看邀请统计中的"成功注册"数量
- 无法区分具体来源（邮件、社交媒体等）
- 建议使用不同的邀请码追踪不同渠道

### Q6: 分享频率有限制吗？

**A**:
- 平台本身没有限制
- 但社交媒体平台可能有发布频率限制
- 建议合理控制分享频率，避免被视为垃圾信息

### Q7: 可以分享到微信或微博吗？

**A**:
- 当前版本不支持直接分享到微信或微博
- 可以复制邀请链接手动分享
- 未来可能会添加更多平台支持

### Q8: 分享后没有人注册怎么办？

**A**:
- 优化分享内容，使其更吸引人
- 选择合适的发布时间
- 增加分享频率
- 尝试不同的社交平台
- 结合邮件邀请等其他方式

### Q9: 可以在分享中添加图片吗？

**A**:
- 取决于社交平台的功能
- Twitter和LinkedIn支持添加图片
- 在分享对话框中可以上传图片
- 建议使用与实验相关的专业图片

### Q10: 社交分享和邮件邀请哪个效果更好？

**A**:
- 各有优势，建议结合使用
- 社交分享：覆盖面广，传播快
- 邮件邀请：针对性强，转化率高
- 根据目标受众选择合适的方式
