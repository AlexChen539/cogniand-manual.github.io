## 主试后台功能：
1. 创建实验
2. 管理实验
3. 管理信息


### 1. 创建实验
**选择创建方式：**

**方式一：使用模板库**
1. 进入"实验工作台"
2. 点击"模板库"
![注册页面](../images/模板库创建实验.png)

3. 浏览并选择合适的实验模板，也可以使用检索功能进行模板筛选

![注册页面](../images/浏览模板库.png)
4. 根据需要修改实验参数
5. 保存并提交审核



**使用 PsyExpGen 工具**
1. 点击"使用 PsyExpGen"。
![注册页面](../images/PsyGen创建实验.png)
2. 在专业工具中设计实验。
![注册页面](../images/PsyGen页面1.png)
3. 填写实验信息。
![注册页面](../images/PsyGen填写基本信息.png)
4. 选择实验框架。
![注册页面](../images/选择框架.png)
5. 填写配置，选择您想使用的api。
![注册页面](../images/填写配置.png)
6. 填写试验方案设计提示词。
![注册页面](../images/填写实验方案设计提示词.png)
7. 代码编辑并且完成实验生成
![注册页面](../images/代码编辑.png)

::: tip 提示
您可以使用左侧的AI助手对话框来辅助设计实验
:::
![注册页面](../images/AI助手.png)
### 等待审核
- 提交后，实验会进入"待审核"状态
- 管理员会审查实验内容和设置
- 通常在 1-3 个工作日内完成审核
- 您会收到审核结果通知

::: warning 注意
如果实验被拒绝，请查看管理员的反馈意见，修改后重新提交。
:::

### 激活实验
审核通过后：
1. 在"我的实验"中找到已批准的实验
2. 点击"激活"开始招募被试
3. 实验会出现在平台的实验目录中
4. 被试可以看到并申请参与




## 管理实验和模板
1. 在后台界面的”我的界面“和”模板管理“查看
![注册页面](../images/实验后台.png)
2. 可以在”我的实验“中查看实验详情和数据
![注册页面](../images/查看实验数据.png)
### 管理参与者
当有被试申请参与时：
1. 在"参与者管理"中查看申请列表
2. 查看申请者的基本信息
3. 决定批准或拒绝申请
4. 批准后，被试可以开始实验
![注册页面](../images/申请管理.png)
::: tip
申请会在 -- 天后自动过期，请及时处理。
:::
## 信息发送

![注册页面](../images/查看收件箱.png)
2. 您可以点击信息卡片查看完整内容
![注册页面](../images/具体内容.png)

3. 点击“写信”按钮，可以编辑信息内容。
![注册页面](../images/写信.png)
