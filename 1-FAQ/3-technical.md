## 文件上传

### Q17: 文件上传失败怎么办？

**A**: 请检查以下几点：

1. **文件大小**
   - 检查文件大小是否超过限制
   - 压缩文件或分批上传

2. **文件格式**
   - 检查文件格式是否正确
   - 转换为支持的格式（CSV、JSON等）

3. **网络连接**
   - 检查网络连接是否稳定
   - 重新上传

4. **存储空间**
   - 联系管理员检查服务器存储空间

**支持的文件格式**：
- CSV（逗号分隔值）
- JSON（JavaScript对象表示法）
- TXT（纯文本）
- ZIP（压缩文件）

详细排查：[故障排除 - 数据上传问题](/6-troubleshooting/3-upload-issues)

### Q18: 页面显示异常怎么办？

**A**: 请尝试以下方法：

1. **刷新页面**
   - Windows: `Ctrl + F5`
   - Mac: `Cmd + Shift + R`

2. **清除缓存**
   - 清除浏览器缓存和Cookie
   - 重新访问平台

3. **无痕模式**
   - 使用浏览器的无痕/隐私模式访问
   - 排除缓存和扩展影响

4. **更新浏览器**
   - 更新浏览器到最新版本
   - 重启浏览器

**清除缓存步骤**：
- Chrome: 设置 → 隐私和安全 → 清除浏览数据
- Firefox: 选项 → 隐私与安全 → Cookie和网站数据 → 清除数据
- Safari: 偏好设置 → 隐私 → 管理网站数据 → 移除全部

## 性能问题

### Q19: 页面加载缓慢怎么办？

**A**: 可能的原因和解决方案：

1. **网络速度慢**
   - 检查网络连接速度
   - 尝试使用更快的网络
   - 避开网络高峰时段

2. **服务器负载高**
   - 避开高峰时段访问
   - 联系管理员检查服务器状态

3. **浏览器缓存过多**
   - 清除浏览器缓存
   - 使用无痕模式访问

4. **数据量过大**
   - 使用分页功能
   - 使用筛选功能减少数据量
   - 减少一次性加载的数据

**优化建议**：
- 关闭不必要的浏览器标签页
- 关闭不必要的浏览器扩展
- 定期清理浏览器缓存
- 使用有线网络代替WiFi

## 邮件接收

### Q20: 未收到验证邮件或重置邮件怎么办？

**A**: 请检查以下几点：

1. **垃圾邮件文件夹**
   - 检查垃圾邮件/垃圾箱
   - 将平台邮箱加入白名单

2. **邮箱地址**
   - 检查注册时填写的邮箱是否正确
   - 重新注册或联系管理员修改

3. **邮件延迟**
   - 等待几分钟后再检查
   - 点击"重新发送"按钮

4. **邮件服务器**
   - 联系管理员检查邮件服务器配置
   - 确认SMTP服务正常
