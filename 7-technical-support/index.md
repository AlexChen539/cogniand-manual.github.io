# 技术支持

需要帮助？我们随时为您提供支持。

## 联系方式

### 技术支持邮箱
**um.andlab@gmail.com**

### 问题反馈渠道
- 平台内反馈功能（登录后在个人中心提交反馈）
- GitHub Issues（如果项目开源）
- 技术支持论坛

### 响应时间
- 工作日：24小时内响应
- 周末和节假日：48小时内响应
- 紧急问题：请在邮件标题注明"紧急"

## 提交问题时请提供

为了更快解决问题，请提供以下信息：

### 1. 问题描述
- 详细描述遇到的问题
- 问题发生的时间和频率

### 2. 操作步骤
- 重现问题的详细步骤
- 预期结果和实际结果

### 3. 环境信息
- 操作系统（Windows/Mac/Linux）
- 浏览器类型和版本
- 账号角色（被试/主试/管理员）

### 4. 错误信息
- 错误提示的完整内容
- 浏览器控制台的错误信息（如有）
- 截图或录屏（如适用）
