通过邮件邀请 ​

概述 ​

邮件邀请是批量邀请被试的高效方式。通过邮件邀请功能，您可以向多个邮箱地址发送个性化的邀请邮件，邮件中包含您的专属邀请链接和自定义消息。这种方式适合向特定的被试群体发送邀请，提高邀请的针对性和转化率。

进入邮件邀请 ​

登录后，进入"邀请用户"页面，在页面右侧找到"通过邮件邀请"区域。

功能说明 ​

1. 添加收件人 ​

1.1 邮箱输入 ​

输入框位置：页面右侧顶部

输入方式：

  • 在邮箱输入框中输入收件人的邮箱地址
  • 支持标准的邮箱格式（如：example@email.com）
  • 输入完成后点击"添加"按钮或按Enter键

邮箱验证：

  • 系统自动验证邮箱格式
  • 无效的邮箱地址无法添加
  • 重复的邮箱地址会被提示

快捷操作：

  • 按Enter键快速添加邮箱
  • 无需每次都点击"添加"按钮
  • 提高批量添加效率

1.2 收件人列表 ​

列表显示：

  • 位于邮箱输入框下方
  • 显示所有已添加的邮箱地址
  • 列表可滚动查看（最大高度32）
  • 显示收件人总数

列表管理：

  • 每个邮箱地址右侧有删除按钮（×）
  • 点击删除按钮移除该邮箱
  • 支持逐个删除或全部清空

收件人数量：

  • 实时显示已添加的收件人数量
  • 格式："{数量} 位收件人"
  • 帮助您掌握邀请规模

2. 编写邀请消息 ​

2.1 消息输入框 ​

位置：收件人列表下方

输入方式：

  • 多行文本框
  • 支持换行和段落
  • 最大长度：300字符

字符计数：

  • 实时显示已输入字符数
  • 格式："{已输入}/{最大长度}"
  • 超过限制无法继续输入

2.2 消息内容建议 ​

推荐内容：

  1. 自我介绍：

    • 简要介绍您的身份和研究背景
    • 说明您的研究机构或团队

  2. 邀请目的：

    • 说明为什么邀请对方参与
    • 强调实验的意义和价值

  3. 实验简介：

    • 简要描述实验内容
    • 说明实验时长和要求

  4. 参与激励：

    • 说明参与实验的好处
    • 如有报酬或奖励，请明确说明

  5. 联系方式：

    • 提供您的联系方式
    • 方便被试咨询和沟通

消息示例：

您好！

我是XX大学心理学系的研究生，正在进行一项关于认知功能的研究。
诚挚邀请您参与我们的在线实验，实验大约需要20分钟。

通过参与实验，您将：
- 了解自己的认知特点
- 为科学研究贡献力量
- 获得实验报告反馈

如有任何疑问，欢迎随时联系我。

期待您的参与！

2.3 消息注意事项 ​

避免内容：

  • 不要包含虚假信息
  • 不要使用误导性的宣传
  • 不要承诺无法兑现的奖励
  • 不要泄露被试的个人信息

语言建议：

  • 使用礼貌、专业的语言
  • 简洁明了，避免冗长
  • 突出重点，吸引注意
  • 真诚友好，建立信任

3. 发送邀请邮件 ​

3.1 发送操作 ​

操作步骤：

  1. 确认收件人列表无误
  2. 检查邀请消息内容
  3. 点击"发送邀请邮件"按钮
  4. 等待发送完成

发送状态：

  • 发送中：按钮显示加载动画
  • 发送成功：显示成功提示
  • 发送失败：显示错误信息

3.2 邮件内容 ​

邮件组成：

  1. 邮件主题：

    • 自动生成，包含"邀请"关键词
    • 吸引收件人打开邮件

  2. 个性化消息：

    • 您编写的自定义消息
    • 显示在邮件顶部

  3. 邀请链接：

    • 您的专属邀请链接
    • 点击即可跳转到注册页面

  4. 平台介绍：

    • CogniAND平台的简介
    • 平台的特点和优势

  5. 注册指引：

    • 简单的注册步骤说明
    • 帮助被试快速完成注册

3.3 发送后处理 ​

自动操作：

  • 清空收件人列表
  • 清空邀请消息
  • 保存邀请记录到本地
  • 更新邀请统计

邀请记录：

  • 记录发送时间
  • 记录收件人邮箱列表
  • 记录邀请消息内容
  • 记录邀请链接

4. 查看邀请记录 ​

4.1 本地记录 ​

存储位置：

  • 邀请记录保存在浏览器本地存储中
  • 记录格式：JSON数据

记录内容：

  • 邀请ID
  • 用户ID
  • 收件人邮箱列表
  • 邀请消息
  • 发送时间
  • 邀请链接

4.2 查看历史 ​

查看方式：

  • 在个人中心查看详细的邀请历史
  • 包括每次邀请的详细信息
  • 可以查看邀请的注册转化情况

详细的邀请历史查看请参考个人中心。

使用建议 ​

1. 选择合适的收件人 ​

目标群体：

  • 符合实验要求的被试
  • 对研究主题感兴趣的人群
  • 有时间和意愿参与的人员

获取邮箱：

  • 通过学术网络获取
  • 通过专业社区获取
  • 通过合作机构获取
  • 确保邮箱来源合法

避免事项：

  • 不要购买邮箱列表
  • 不要发送垃圾邮件
  • 不要向不相关的人发送
  • 不要频繁发送重复邮件

2. 优化邀请消息 ​

提高打开率：

  • 使用吸引人的邮件主题
  • 在消息开头说明重点
  • 保持消息简洁明了

提高转化率：

  • 说明参与的价值和意义
  • 提供清晰的行动指引
  • 强调实验的简单和便捷
  • 提供联系方式建立信任

个性化建议：

  • 根据收件人特点调整消息
  • 使用友好、专业的语气
  • 避免使用模板化的语言

3. 合理控制发送频率 ​

发送时机：

  • 选择合适的发送时间
  • 避免在深夜或节假日发送
  • 考虑收件人的时区

发送频率：

  • 不要频繁发送邮件
  • 给收件人足够的考虑时间
  • 如需提醒，间隔至少一周

跟进策略：

  • 对未回复的收件人适当跟进
  • 跟进邮件要简短礼貌
  • 尊重对方的选择

4. 遵守邮件营销规范 ​

法律法规：

  • 遵守反垃圾邮件法规
  • 提供退订选项（如适用）
  • 保护收件人隐私

平台规则：

  • 遵守CogniAND平台使用条款
  • 不得进行虚假邀请
  • 不得滥用邮件功能

注意事项 ​

⚠️ 重要提示：

  1. 邮箱来源

    • 确保邮箱来源合法
    • 不要使用非���获取的邮箱列表
    • 尊重收件人的隐私权

  2. 发送限制

    • 单次发送建议不超过50个邮箱
    • 避免短时间内大量发送
    • 防止被识别为垃圾邮件

  3. 邮件内容

    • 不要包含虚假信息
    • 不要使用误导性的宣传
    • 不要承诺无法兑现的奖励

  4. 隐私保护

    • 不要在邮件中泄露其他被试的信息
    • 使用密送（BCC）方式发送（如适用）
    • 保护收件人的邮箱地址

  5. 垃圾邮件风险

    • 避免使用垃圾邮件常用词汇
    • 不要使用全大写或过多感叹号
    • 提供真实的发件人信息

常见问题 ​

Q1: 一次可以发送多少封邮件？ ​

A:

  • 建议单次发送不超过50个邮箱
  • 过多可能被识别为垃圾邮件
  • 如需大量发送，请分批进行

Q2: 邮件发送失败怎么办？ ​

A:

  • 检查收件人邮箱地址是否正确
  • 检查网络连接是否正常
  • 稍后重试或联系技术支持

Q3: 收件人没有收到邮件怎么办？ ​

A:

  • 提醒收件人检查垃圾邮件文件夹
  • 确认邮箱地址是否正确
  • 确认邮件服务器是否正常
  • 可以尝试重新发送

Q4: 可以撤回已发送的邮件吗？ ​

A:

  • 不可以，邮件发送后无法撤回
  • 发送前请仔细检查内容
  • 如有错误，可以发送更正邮件

Q5: 邀请消息可以包含链接吗？ ​

A:

  • 可以，但建议只包含必要的链接
  • 过多链接可能被识别为垃圾邮件
  • 邀请链接会自动包含在邮件中

Q6: 如何提高邮件打开率？ ​

A:

  • 使用吸引人的邮件主题
  • 选择合适的发送时间
  • 确保邮件内容相关且有价值
  • 避免被识别为垃圾邮件

Q7: 可以使用邮件模板吗？ ​

A:

  • 可以，但建议根据收件人调整
  • 避免使用完全相同的模板
  • 个性化的邮件效果更好

Q8: 邮件发送记录会保存多久？ ​

A:

  • 记录保存在浏览器本地存储中
  • 清除浏览器数据会删除记录
  • 建议定期备份重要记录

Q9: 可以向同一个邮箱多次发送邀请吗？ ​

A:

  • 可以，但不建议频繁发送
  • 给收件人足够的考虑时间
  • 过度发送可能被视为骚扰

Q10: 邮件邀请和链接分享有什么区别？ ​

A:

  • 邮件邀请：主动发送给特定收件人，更有针对性
  • 链接分享：被动等待被试点击，覆盖面更广
  • 两种方式可以结合使用，提高邀请效果
